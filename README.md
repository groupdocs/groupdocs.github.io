# GroupDocs on GitHub — product index

GitHub Pages site served at <https://groupdocs.github.io/> — the index of all
GroupDocs open-source example repositories (each card links to a
`groupdocs-<product>.github.io` landing page).

## Maintenance
- **`index.html` is generated — do not edit it by hand.** It is built from a
  shared design with live GitHub stars and versions
  (<https://products.groupdocs.com/versions.json>).
- `groupdocs.svg` is the logo / icon asset.
- **`img/` — keep it.** These images are referenced **externally** (not from
  `index.html`), so do not delete the folder even though the generated page does
  not link to it. (The old `css/` and `fonts/` from the previous design were
  removed; `img/` must stay.)
